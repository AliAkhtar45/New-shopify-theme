# Health Geeks Shopify Theme

This is a beginner-friendly Shopify Online Store 2.0 theme starter based on the dark Health Geeks design.

## What is included

- `layout/theme.liquid`
- JSON templates for home, product, collection, cart, and contact page
- sections for header, footer, hero, featured collection, product, collection, cart, and contact form
- a shared stylesheet and small mobile-menu script

## How to use it in Shopify

1. In Shopify admin, go to `Online Store > Themes`.
2. Click `Add theme`.
3. Upload the `health-geeks-shopify-theme` folder as a zip, or use Shopify CLI.
4. After upload, click `Customize`.
5. Set your logo in theme settings.
6. Create a navigation menu and assign it to the header/footer sections.
7. Add your products in Shopify.
8. Pick your featured product and collection in the theme editor.

## Important note

Shopify handles the real checkout, address fields, payments, and orders. This theme handles the storefront design.
